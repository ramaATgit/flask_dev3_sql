CSV Upload

Upload a CSV file to update account balances.

Upload CSV File
The CSV file should contain columns: Account, bal
CSV Format Requirements

Your CSV file must meet the following requirements:

  • Must have Account and bal columns
  • Account names must match existing accounts in the system
  • Balance (bal) should be a valid number
Example CSV Format:
Account,bal
Current Account,1500.25
Savings Account,3000.00
ISA Account,5250.75
The system will create a backup of account data before updating, which can be viewed in the reports section.
